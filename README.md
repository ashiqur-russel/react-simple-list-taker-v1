# Getting Started with Simple List Taking React App


# Used React hook for this app
  `useState` & `useReduce`
  
## Functionality 
  1. Add Item.
  2. Delete Item.
  3. Modal show while while deleting , adding or try to enter empty item.

Live Site : https://react-list-taker.netlify.app/
